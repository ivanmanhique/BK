# BookingSystem
## A **user** will be able to:
1. Log in the application
2. Choose a destination
3. Look for an Hotel
4. Book the hotel
5. Cancel a booking
   

## Backend:
1. Store users data in a database, data such as:
    Destination
    Date of visit
    Booked Hotel
    Opinion of Hotel
    Ratings
2. Organize Hotels according to its ratings & prices.
